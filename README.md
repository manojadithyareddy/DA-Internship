# 📊 ZenithRetail Sales & Profitability Recovery

## Project Overview

This project analyzes retail sales performance using SQL, Python, and Power BI to identify profit-draining products, optimize discounts, and generate executive business insights.

---

## Business Problem

ZenithRetail experienced declining profitability despite increasing sales. The objective was to identify loss-making products, evaluate customer behavior, and develop an executive dashboard to support data-driven decision-making.

---

## Objectives

- Clean and validate retail sales data
- Perform SQL-based business analysis
- Conduct Exploratory Data Analysis (EDA) using Python
- Build RFM customer segmentation
- Create an interactive Power BI dashboard
- Forecast monthly profit trends

---

## Key Business Insights

- Technology generated the highest profit.
- Furniture showed the lowest profitability.
- High discounts reduced profit margins.
- West Region contributed the highest revenue.
- VIP customers generated most sales.
- Profit forecast indicates continued business growth.
